O HTML

O HTML é uma linguagem de marcação para internet.
È a linguagem "base" da internet, necessária para definir
onde cada elememto (texto,imagem,video,etc) deve estar.

O navegador (Crhrome , IE , Firefox , Opera , Safari) "le" o código digitado e o "interpreta" exibindo assim na tela o site
ja formatado e não códigos e TAGS.

O CSS

Folha de estilo em cascata , serve para formatar as TAGS definidas no HTML.

As Formatações incluem cor ,alinhamento,altura,largura, aparencia de objetos em geral.

Principais TAGS

  • A TAG <p> serve para definir um parágrafo
  • A TAG <h1>serve para definir a importancia do titulo
  • A TAG <br> serve para querbrar uma linha
  • A TAG <code> serve para definir uma area de código
  • A TAG <ul> serve para definir uma lista
  • A TAg <li> serve para respresntar um item de uma lista ,usada juntamente com a <>
